##Як запустити frontend
- Відкрити файл index.html у браузері.
##Що перевіряти
- Чи коректно завантажується сторінка.
- Чи працює форма додавання інцидентів.
- Чи відображається список інцидентів.
- Чи працюють пошук, фільтр, сортування, редагування та видалення.
- Чи зберігаються дані після перезавантаження сторінки (localStorage).
#Рівень виконання
- Реалізовано повноцінний клієнтський застосунок для трекінгу інцидентів кібербезпеки.
- Створено структуру репозиторію та організовано код у логічні модулі (state, render, handlers).
- Додано клієнтську валідацію всіх полів форми з повідомленнями про помилки.
- Реалізовано додавання, редагування та видалення інцидентів.
- Додано пошук, фільтрацію та сортування списку.
- Застосовано делегування подій для елементів списку.
- Забезпечено збереження даних у localStorage (персистентність).
- Оформлено інтерфейс із базовим CSS для читабельності та зручності.
