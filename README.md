## Як запустити frontend
- Відкрити файл `frontend/index.html` у браузері.

## Що перевіряти
- Чи відкривається HTML‑сторінка.
- Чи відображається базовий контент (наприклад, заголовок або текст “Працює!”).

## Рівень виконання
- Створено структуру репозиторію.
- Реалізовано smoke‑check для frontend.
